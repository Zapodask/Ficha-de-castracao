# Ficha de castração

Nextjs + Pdflib fill pdf form

## Installation

Use [yarn] (https://classic.yarnpkg.com/en/docs/install/) to install packages

```bash
yarn install
```

## Usage

```bash
#Development
yarn dev

#Build
yarn build

#Format
yarn format

#Test
yarn test

#Test all
yarn test-all
```
